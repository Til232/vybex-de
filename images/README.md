## How to Download

1. Visit [Unsplash.com](https://unsplash.com)
2. Search for the image using keywords above
3. Click on the image
4. Download at desired size
5. Save to this folder with the recommended filename

## Alternative: Using URLs

If you prefer not to download, the website currently uses direct Unsplash URLs which work fine for production.

## Image Optimization

After adding images, consider:
- Compressing with [TinyPNG](https://tinypng.com)
- Converting to WebP format for better performance
- Using responsive image sizes
